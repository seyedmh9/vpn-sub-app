# ✅ نسخه نهایی - VPN Subscription با هسته sing-box

این نسخه **واقعی** است و از هسته قدرتمند **sing-box** استفاده می‌کند.

## ویژگی‌ها
- دانلود خودکار سابسکریپشن
- پشتیبانی از پروتکل‌های: VLESS • VMess • Trojan • WireGuard
- اتصال واقعی و پایدار
- رابط کاربری فارسی

---

## نحوه استفاده

### 1. سابسکریپشن خود را قرار دهید

در فایل `MainActivity.kt` فقط این خط را تغییر دهید:

```kotlin
private val SUBSCRIPTION_URL = "https://YOUR-SUBSCRIPTION-URL"
```

### 2. بیلد و اجرا

1. پروژه را در **Android Studio** باز کنید
2. Gradle Sync بزنید
3. Run کنید

---

## ساختار فایل‌های مهم

- `MainActivity.kt` → دانلود ساب + رابط کاربری
- `SingBoxManager.kt` → مدیریت هسته sing-box + تبدیل کانفیگ
- `VpnServiceImpl.kt` → سرویس VPN اندروید
- `Config.kt` → مدل کانفیگ

---

## نکات مهم

- sing-box به صورت خودکار کانفیگ را parse می‌کند
- برای پروتکل‌های پیچیده‌تر (مثل Reality) می‌توانید تابع `parseVless` را گسترش دهید
- برنامه به صورت خودکار تونل TUN راه می‌اندازد
