# راهنمای کامل ساخت APK با GitHub Actions (بدون اندروید استودیو)

این راهنما قدم به قدم توضیح می‌دهد چطور بدون نصب هیچ برنامه‌ای، APK برنامه VPN Subscription را بسازید.

---

## مرحله ۱: ساخت حساب گیت‌هاب (اگر نداری)

1. به آدرس [github.com](https://github.com) برو
2. روی **Sign up** کلیک کن و حساب بساز (رایگان است)

---

## مرحله ۲: ساخت مخزن جدید (Repository)

1. بعد از ورود به گیت‌هاب، روی دکمه سبز **New** (بالا سمت چپ) کلیک کن
2. نام مخزن را بگذار: `vpn-sub-app`
3. تیک **Add a README file** را بزن
4. روی **Create repository** کلیک کن

---

## مرحله ۳: آپلود فایل‌های پروژه

### روش A: با گوشی (ساده‌ترین)

1. مخزن را باز کن
2. روی دکمه **Add file** → **Upload files** کلیک کن
3. تمام فایل‌ها و پوشه‌های داخل `vpn-sub-app` را انتخاب و آپلود کن:
   - پوشه `app`
   - فایل `build.gradle`
   - پوشه `.github`
   - و بقیه فایل‌ها
4. پایین صفحه روی **Commit changes** کلیک کن

### روش B: با کامپیوتر (اختیاری)

```bash
git clone https://github.com/USERNAME/vpn-sub-app.git
# تمام فایل‌ها را کپی کن داخل پوشه
git add .
git commit -m "Initial commit"
git push
```

---

## مرحله ۴: اضافه کردن سابسکریپشن خودت

قبل از آپلود، حتماً سابسکریپشن خودت را در فایل `MainActivity.kt` قرار بده:

```kotlin
private val SUBSCRIPTION_URL = "https://YOUR-SUBSCRIPTION-URL"
```

---

## مرحله ۵: ساخت خودکار APK

بعد از آپلود فایل‌ها، گیت‌هاب به صورت خودکار APK را می‌سازد:

1. برو به تب **Actions** در مخزن گیت‌هاب
2. روی **Build APK** کلیک کن
3. روی دکمه سبز **Run workflow** کلیک کن
4. صبر کن تا تمام شود (معمولاً ۳-۵ دقیقه طول می‌کشد)

---

## مرحله ۶: دانلود APK

وقتی ساخت تمام شد:

1. به تب **Actions** برو
2. روی آخرین اجرا (Run) کلیک کن
3. پایین صفحه بخش **Artifacts** را پیدا کن
4. روی **VPN-Subscription-APK** کلیک کن تا دانلود شود

فایل دانلود شده `app-debug.apk` است.

---

## مرحله ۷: نصب روی گوشی

1. فایل `app-debug.apk` را به گوشی منتقل کن
2. روی گوشی نصب کن (ممکن است نیاز به فعال کردن "نصب از منابع ناشناس" داشته باشی)
3. برنامه را باز کن

---

## نکات مهم

- هر بار که سابسکریپشن را تغییر می‌دهی، فقط فایل `MainActivity.kt` را آپلود کن و دوباره **Run workflow** بزن.
- APK ساخته شده در بخش Artifacts تا ۳۰ روز نگه داشته می‌شود.
- اگر خطایی دیدی، در تب Actions لاگ را چک کن.
