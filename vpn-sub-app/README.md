# VPN Subscription Android App

یک اپلیکیشن اندروید ساده برای مدیریت سابسکریپشن‌های VPN (مثل V2Ray, Clash و غیره) و اتصال به کانفیگ‌ها.

**نکته مهم:** این یک اسکلت کامل و آماده برای استفاده در Android Studio است. برای اتصال واقعی VPN نیاز به ادغام کتابخانه‌هایی مثل v2rayNG یا sing-box دارید (کدها و راهنما در ادامه).

## ویژگی‌ها
- افزودن سابسکریپشن (URL)
- نمایش لیست کانفیگ‌های هر ساب
- اتصال و قطع اتصال VPN (با VpnService پایه)
- رابط کاربری فارسی

## نحوه استفاده
1. پروژه را در Android Studio باز کنید
2. Gradle Sync بزنید
3. اپ را Build و Run کنید
4. برای اتصال واقعی، کد VpnService را با کتابخانه مناسب جایگزین کنید

## ساختار پروژه
- MainActivity: صفحه اصلی
- VpnService: سرویس VPN
- Models: Subscription و Config

## افزودن قابلیت واقعی VPN
برای پشتیبانی واقعی از کانفیگ‌های V2Ray:
- از پروژه open-source v2rayNG الهام بگیرید
- یا از کتابخانه‌های آماده مثل `com.github.2dust.v2rayNG` استفاده کنید
