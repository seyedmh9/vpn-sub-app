# VPN Subscription - نسخه واقعی با اتصال

این نسخه شامل موارد زیر است:

- سابسکریپشن **هاردکد شده** داخل کد
- دانلود خودکار کانفیگ‌ها هنگام باز شدن برنامه
- اتصال **واقعی** با Android VpnService
- رابط کاربری فارسی

## نحوه استفاده

1. آدرس سابسکریپشن خود را در فایل `MainActivity.kt` در خط `SUBSCRIPTION_URL` قرار دهید
2. برنامه را در اندروید استودیو بیلد کنید
3. برای اتصال واقعی، کانفیگ‌ها باید از نوع **WireGuard** یا **VLESS** ساده باشند

**نکته:** برای پشتیبانی کامل از V2Ray/VLESS/VMess نیاز به ادغام هسته sing-box یا v2ray-core است.

## سابسکریپشن نمونه

سابسکریپشن شما باید به صورت base64-encoded باشد و هر خط یک کانفیگ باشد:
```
dmxlc3M6Ly9kZW1v...
```

---

## نحوه تغییر سابسکریپشن

در فایل `MainActivity.kt` این خط را تغییر دهید:

```kotlin
private const val SUBSCRIPTION_URL = "https://YOUR-SUBSCRIPTION-URL-HERE"
```
